Produces a .xml file with record entries for use with Odoo Accounting.

Requirements to run:
    Python 3

Step-by-step:
    1. Check data in Data folder if valid.
    2. (Optional) Edit Generator.py and change idcount to set what id number it starts with.
    3. Run Generator.py

All data regarding Philippine Accounting in data folder courtesy of Bureau of Internal Revenue (BIR) Account Information Form 1702AIF 
  http://www.bir.gov.ph/images/bir_files/old_files/pdf/19201702aif.pdf
